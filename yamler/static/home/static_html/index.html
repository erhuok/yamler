  • 我们是怎样工作的?

    可汗学院（Khan Academy）成立于2007年，通过互 联网，免费提供优质的教学资源。与传统课堂不同 的是，"可汗学院"课程通常只有10～15分钟， 从最基础的内容开始，以由易到难的进阶方式互相 衔接。目前在网易上线的已翻译课程达到300集， 内容包括数学，化学，生物学等。这些课程可与 美国的SAT考试相衔接。
